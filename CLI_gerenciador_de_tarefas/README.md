# Gerenciador de Tarefas (CLI)

Projeto simples em **Python** de um **gerenciador de tarefas via linha de comando (CLI)**, desenvolvido para praticar lógica de programação, estruturas de dados, funções e interação com o usuário.

---

## Funcionalidades

- Adicionar tarefas
- Listar todas as tarefas
- Remover tarefas
- Marcar tarefas como concluídas ou pendentes
- Filtrar tarefas por status:
  - Pendentes
  - Concluídas
- Salvar tarefas em um arquivo `.txt`
- Tratamento de entradas inválidas do usuário

---

## Tecnologias utilizadas

- Python 3
- Execução via terminal (CLI)

---

## Comandos disponíveis

- add: Adicionar nova tarefa
- list: Listar tarefas
- remove: Remover tarefa pelo índice
- check: Marcar/desmarcar tarefa como concluída
- status: Filtrar tarefas por status
- save: Salvar tarefas em arquivo .txt
- close: Encerrar o programa
