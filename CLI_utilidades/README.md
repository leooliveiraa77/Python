# Car Dealership CLI Toolkit

Um CLI (Command Line Interface) desenvolvido em Python para simular um sistema simples de gerenciamento de uma concessionária de veículos.  
O projeto foi criado com foco em prática de Python, manipulação de arquivos, uso de `argparse` e organização de dados.

## Funcionalidades

O sistema permite executar as seguintes ações via terminal:

- Saudação personalizada usando argumentos de linha de comando
- Adicionar carros ao estoque
- Listar carros:
  - todos
  - ordenados por ano
- Salvar os dados em arquivo JSON
- Converter o arquivo JSON para CSV
- Validar CPF ou CNPJ para simular a compra de um carro

## Tecnologias Utilizadas

- Python 3
  - `argparse`
  - `json`
  - `csv`
  - `pathlib`
